# Logo Banner

Standalone script that injects a Knight Witch logo/banner section.

## Files

- `logo-banner.js`

## Ownership

`logo-banner.js` creates and inserts the banner wrapper, media container, and media element around the current script placement.

## Asset

```text
https://knightwitch.nyc3.cdn.digitaloceanspaces.com/BANNERS/Knight%20Witch%20Redux%202.webm
```

## Classes

```text
.kw-logo-banner-wrapper
.kw-logo-banner

.kw-logo-banner-video
```

## Status

Standalone. Not documented as a dependency of the global Fourthwall loader.
