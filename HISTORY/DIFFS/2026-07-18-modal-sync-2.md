# 2026-07-18 — Unified standard and Step 3 product modal presentation

## Scope

Synchronize the visual presentation of the standard Spellweave/Cauldron Core `kwfw` expanded product modal and Step 3 jacket `kwpj` modal without merging or changing their product-loading, variant, pricing, cart, or gallery-selection implementations.

## Runtime changes

### `fourthwall/kw-product-modal-presentation.css`

Added one shared visual layer for both modal namespaces.

Surface and typography changes:

- Sets panel, panel grid, gallery, track, and information column to black.
- Applies AgencyFB to product titles, prices, labels, selects, quantity controls, buttons, and descriptions.
- Uses white titles, red prices, red collection subtitles, orange primary CTA, dark secondary CTA, and dark gray form controls.
- Standardizes circular close controls.

Desktop geometry:

```text
panel width: min(1120px, 96vw)
grid columns: minmax(380px, 1.18fr) minmax(360px, .82fr)
gallery height: 540px
track height: 540px
information padding: 46px 30px 30px
```

Media behavior:

- Both systems use full-height gallery slides.
- Images/videos use `object-fit:contain` and `object-position:top center`.
- Media backgrounds remain black.
- Single-media galleries hide arrows, dots, and the standard swipe cue.

Navigation changes:

- Replaces Step 3 boxed arrows with the standard transparent white chevron treatment.
- Desktop arrows use `44px × 60px`, `10px` edge offsets, and centered vertical placement.
- Mobile arrows use `38px × 56px`, `8px` edge offsets, and the same transparent treatment.
- Both dot systems share size, inactive opacity, and orange active state.

Mobile geometry:

```text
modal padding: 8px
panel width: min(100%, 560px)
panel max-height: calc(100dvh - 16px)
gallery/track height: clamp(360px, 118vw, 620px)
information padding: 28px 22px 24px
```

This replaces the standard modal’s media-driven height expansion that produced excessive blank space below mobile imagery.

Collection-title styles:

- Adds `.kw-product-collection-link` as a smaller red AgencyFB subtitle.
- Adds duplicated pseudo-elements and a `kwProductCollectionGlitch` keyframe matching the site navigation’s sliced text displacement treatment.
- Provides a static reduced-motion fallback.

### `fourthwall/kw-product-modal-presentation.js`

Added controlled title formatting and collection navigation for both modal namespaces.

Behavior:

- Reads the original title from the modal product object without mutating it.
- Recognizes `Cyberpunk 2077` as a prefix or suffix separated by dash, em dash, en dash, colon, or pipe.
- Leaves unrelated titles unchanged.
- Replaces the displayed title with the clean primary product name.
- Creates a red linked subtitle pointing to `/pages/edgerunners`.
- Desktop mouse/focus interaction glitches between `Cyberpunk 2077` and `Edgerunners Collection`.
- Mobile alternates between the two labels every four seconds.
- Reduced-motion mode disables automatic cycling and animated glitch transitions.
- Uses WeakMap-owned timers and stops mobile cycles when links disconnect.
- Uses signature-based idempotence so the MutationObserver does not reset subtitle state on its own DOM mutations.

## Architecture boundary

Unchanged owners:

- Standard product loading/modal construction/cart: production-pinned `kwfw-carousel.js`.
