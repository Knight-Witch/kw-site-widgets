# Media Notes

Native product media returned by the Fourthwall product API can remain on Fourthwall.

All other site media used by this repo is on the Knight Witch DigitalOcean CDN unless a documented exception says otherwise.

That includes banners, background videos, page videos, product-support media, feature cards, icons, SVGs, fonts, logos, and other brand or interface media.

CDN root:

```text
https://knightwitch.nyc3.cdn.digitaloceanspaces.com
```

Repo-local assets belong in `/ASSETS/`.
